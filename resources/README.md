# Resources

Place icon files here before building:

- `icon.ico` — Windows icon (256x256)
- `icon.icns` — macOS icon bundle
- `icon.png` — Linux + tray icon (256x256 PNG)
- `tray-icon.png` — System tray icon (16x16 or 32x32 PNG)

These are gitignored. Source artwork lives in `companion-web/public/brand/`.
